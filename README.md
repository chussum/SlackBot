# SlackBot
    ECMAScript2015 (ES6)
    
# Getting Started
#### 1. Install yarn / pm2
    1-1. npm install -g yarn
    1-2. npm install -g pm2
#### 2. Modify the server.js file
    vi ./server/server.js
#### 3. Run Slack Bot
    for development
      $ yarn dev
    for production (using pm2)
      $ yarn start
